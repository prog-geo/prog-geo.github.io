Avaliação de recorrência de cobertura de nuvens no BDC

Link para os scripts gerados pelo grupo.

Os aplicativos de Big Data de Observação da Terra são ferramentas poderosas para aprimorar o armazenamento, a organização e o gerenciamento de dados de observação da Terra em grande escala. Um exemplo é a plataforma Brazil Data Cube (BDC), que fornece cubos de dados de média resolução espacial para o território brasileiro. Com base na coleção Sentinel-2 Nível 2A disponível no BDC, esse trabalho deve gerar uma avaliação rápida da dinâmica da cobertura de nuvens, para alguma região e um período de interesse, contendo informações estatísticas básicas e visualizações.

Observações

1. Os scripts deverão conter documentação de uso.

2. O relatório produzido deverá estar no formato do evento GEOINFO.
