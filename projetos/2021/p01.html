Brazil Data Cube Cloud Coverage (BDC3)¶

O trabalho apresentado em [10] introduz uma aplicação para fornecer ao usuário algumas estatísticas sobre a cobertura de nuvens a partir de séries temporais de imagens de sensoriamento remoto ópticas. Esta aplicação utiliza o serviço STAC do Brazil Data Cube para recuperar, nos metadados das coleções de imagens e itens, o percentual de cobertura de nuvens e o footprint das imagens (Figura % s). Assim, vários gráficos podem ser apresentados para o usuário, como porcentagem de cobertura de nuvens mensal ou anual, ou valores absolutos da cobertura de nuvem ao longo do tempo.

Arquitetura do BDC3.
Fonte: Lucena et al. (2020) [10].¶

Para este projeto, considere criar um ou mais pacotes Python que implementem um processador de dados matriciais e vetoriais a partir das mácaras de nuvem disponíveis no catálogo do BDC, oferecendo a possibilidade de estatísticas a partir de partições arbitrárias, como limites municipais e estaduais, grade de um dado satélite/sensor, limite de um bioma, entre outros. Considere criar Notebooks que ajudem a apresentar as funcionalidades desse processador.

Nota

1. Os produtos de código desenvolvidos deverão ser licenciados de acordo com os modelos de software livre MIT, e disponibilizados em um ou mais repositório na organização prog-geoespacial.

2. Os pacotes deverão conter documentação de uso.

3. Utilize o template de pacotes Python do Brazil Data Cube.

4. Os produtos de dados de exemplo deverão ser disponibilizados de acordo com o modelo Creative Commons.

5. Produzir um relatório no formato da GeoInfo.
