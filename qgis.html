6. Programando com o QGIS

O aplicativo geográfico QGIS possui uma forte integração com a linguagem Python, permitindo utilizá-la de diversas maneiras. O console Python, disponível de dentro da interface gráfica do QGIS (Figura 6.1), permite que o usuário entre com comandos válidos da linguagem Python.

Também é possível criar scripts que utilizam a API do QGIS em Python para automação de tarefas tanto na inicialização do QGIS quanto para realização de processamentos em lote, como mostrado na Figura 6.2.

Por último, é possível estender as funcionalidades do QGIS através da criação de novos plugins escritos na linguagem Python. O plugin Web Land Trajectory Service (WLTS), mostrado na Figura 6.3, é desenvolvido nessa linguagem e acrescenta ao QGIS funcionalidades para extração de trajetórias a partir de conjuntos de dados de uso e cobertura da Terra.

Este capítulo irá apresentar algumas maneiras de utilizar a linguagem Python em conjunto com o aplicativo QGIS.
