PyPI
====


.. image:: ../../img/variados/pypi-logo.svg
    :alt: Python Package Index
    :width: 128
    :align: right


O `PyPI <https://pypi.org/>`_, ou *Python Package Index*, é um dos principais repositórios de pacotes da linguagem Python. Trata-se do repositório padrão utilizado pela ferramenta de gerenciamento de pacotes `pip <https://github.com/pypa/pip>`_ quando não especificado o repositório a ser usado.
