Atualizando um Pacote
*********************


Podemos também utilizar o conda para atualizar um pacote em nosso ambiente de trabalho corrente. Para isso, podemos usar o comando ``conda update``, como no exemplo abaixo:


.. code-block:: shell

    conda update gdal
