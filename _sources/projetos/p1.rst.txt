Projeto 1
---------


Usando um serviço de consulta a séries temporais de sensoriamento remoto, como o `WTSS <https://github.com/brazil-data-cube/wtss.py/blob/master/examples/wtss.ipynb>`_, coletar várias séries temporais e calcular métricas (através do pacote Python `stmetrics <https://stmetrics.readthedocs.io/>`_ por exemplo). Baseado em análises empíricas, verificar qual conjunto de métricas pode servir para mapear séries temporais com comportamento temporal equivalente a alvos de água.


Para facilitar a organização, utilizar as áreas de interesse disponíveis no projeto `Brazil Data Cube <http://brazildatacube.org/>`_ com dados dos satélites disponíveis (``CBERS-4``, ``Landsat-8``, ``Sentinel-2``).
