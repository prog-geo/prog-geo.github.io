Projeto 3
---------


Desenvolver uma ferramenta para geração de índices espectrais. Considerando uma lista de arquivos e alguns parâmetros, este projeto envolve a criação de uma ferramenta para linha comando do tipo:


.. code-block:: shell

    spectral --index ndvi --red red.tif --nir nir.tif --output ndvi.tif --verbose

    spectral --index all --red red.tif --green green.tif --blue blue.tif --nir nir.tif ...  --output-dir --suffix "_all"
