Projeto 6
---------


Preparação de uma máscara de nuvens composta a partir de diversos métodos (`Sen2Cor <https://step.esa.int/main/third-party-plugins-2/sen2cor/>`_, `Maja <https://labo.obs-mip.fr/multitemp/maccs-how-it-works/>`_, `fmask <http://www.pythonfmask.org/>`_, etc.). O usuário do sistema pode escolher diferentes maneiras de fazer a composição: ``maioria``, ``intersecção``, ``união``, etc. Apresentar uma estatística básica com o sumário de cada método aplicado.


Considerando uma lista de arquivos e alguns parâmetros, este projeto envolve a criação de uma ferramenta para linha comando do tipo:


.. code-block:: shell

    cloud-mask-compose --input (1 ou mais arquivos?) --methods "Sen2Cor, Maja, FMask" --strategy "majority" --output mask-file.tiff
