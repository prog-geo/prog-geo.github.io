Serviço web de registro de novas coleções/itens no STAC da BIG
==============================================================


Atualmente, a BIG conta com um catálogo STAC dinâmico, formado por um serviço Python integrado a um sistema de bancos de dados PostgreSQL com a extensão PostGIS. Este trabalho tem como objetivo construir um serviço Web que permita realizar a criação de novas coleções, atualização de metadados de coleções, inserção/atualização de novos itens das coleções, considerando o privilégio de usuários. Este projeto poderá ser desenvolvido sob orientação da equipe da BIG/BDC.
