Projeto 4
---------


Desenvolver um gerador de ``Modelo Linear de Mistura`` para uma série temporal, dado um conjunto de pontos de amostra preexistente. Considerando uma lista de arquivos e alguns parâmetros, este projeto envolve a criação de uma ferramenta para linha comando do tipo:


.. code-block:: shell

    lmm-cli --samples arq.json --b1 b1.tiff --b2 b2.tiff --b3 b3.tif --output-dir "dir" --randomize-samples
