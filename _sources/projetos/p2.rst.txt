Projeto 2
---------


Utilizando um conjunto de imagens ``CBERS-4``, descobrir quais imagens são da mesma região, e aplicar um algoritmo simples de detecção de mudanças, por exemplo (:math:`img_2 - img_1`). Quando houver uma quantidade de mudanças superior a um dado limiar (informado pelo analista), apresentar um alerta na região encontrada.


Para realizar este trabalho, utilize o catálogo de coleções de imagens do projeto `Brazil Data Cube <http://brazildatacube.org/>`_. Um exemplo de uso de uma biblioteca ``Python`` para consulta e download das imagens contidas nesse catálogo pode ser vista em `Python Client Library for STAC <https://github.com/brazil-data-cube/stac.py>`_ e no Jupyter Notebook `stac.ipynb <https://github.com/brazil-data-cube/stac.py/blob/master/examples/stac.ipynb>`_.
