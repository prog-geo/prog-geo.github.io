6.2. Terminal Interativo Python

Um interpretador é um programa capaz de executar outros programas. A forma mais simples de rodar programas escritos em Python é através do seu terminal interativo (Figura 6.13).

Para acessar o interpretador de comandos do Python na distribuição Anaconda, na barra de atalhos de programas do Windows, localize o grupo Anaconda e, em seguida, escolha a opção Anaconda Prompt, conforme Figura 6.10.

Na janela aberta, digite o comando python, conforme mostrado na Figura 6.11.

No macOS, abra um terminal de comandos e digite o comando python, isto deverá abrir o terminal de comandos como mostrado na Figura 6.12.

No Linux, abra um terminal de comandos e digite o comando python (ou python3), isto deverá abrir o terminal de comandos como mostrado na Figura 6.13
